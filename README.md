# AdventureWorks-PowerBI-Dashboard
Interactive Power BI dashboard built using AdventureWorksDW2019

## Overview

This project is an interactive Power BI dashboard built using the AdventureWorksDW2019 data warehouse. The dashboard provides insights into sales performance, customer behavior, product profitability, and geographic distribution.

## Dashboard Pages

* Executive Summary
* Customer Analytics
* Product Performance
* Geographic Analysis

## Key Features

* Interactive filtering and navigation
* KPI cards and performance indicators
* Profit and margin analysis
* Customer segmentation
* Geographic sales analysis
* Custom tooltips and bookmarks
* Responsive dashboard design

## Tools Used

* Power BI Desktop
* DAX
* SQL Server (AdventureWorksDW2019)
* GitHub

* `Dashboard.pbix` – Power BI project file
